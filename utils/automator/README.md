# 基于强化学习的自动探索模块说明

## 环境说明

### 观察空间

一维数组

初始大小为2000

两部分组成：应用总activity数大小的one-hot向量 和 当前activity包含的控件数大小的one-hot向量

### 动作空间

二维数组

每个动作包含控件、具体动作两个值



## 算法说明

### 随机算法

正常

### Q-Learn

存在控件数量爆炸问题（后续可能通过归类方法解决）

### SAC

正常



## 挑战
