# DPC_tools
APP隐私合规性检测工具

## 运行方法

解压static中的static.7z压缩包中的文件到static目录

运行main.py文件即可

## 动态检测

### 检测项目

- [ ] 是否有隐私政策弹窗
- [ ] 未同意隐私政策时是否有获取用户隐私的行为
- [ ] 同意隐私政策后是否按照政策使用隐私信息
